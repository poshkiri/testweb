Редактирование каталога

С чего начать (коротко)

  1. Получите секретный ключ у того, кто настраивал сайт (разработчик / администратор). Сам ключ прописывается на сервере в файле настроек, обычно это не ваш уровень — просто попросите готовый ключ.
  2. Откройте эту страницу в браузере (закладку вам дадут или пришлют ссылку вида …/admin.html).
  3. Вставьте ключ в поле ниже и нажмите «Загрузить с сервера» — подгрузится текущий список блюд.
  4. Чтобы добавить блюдо: в большом поле найдите блок одного товара (между { и }), скопируйте его целиком, вставьте копию ниже через запятую. В копии поменяйте id (уникальный, например p15), name, price, unit, при необходимости ссылку на фото image.
  5. Нажмите «Сохранить», затем откройте витрину каталога и обновите страницу (F5), чтобы увидеть изменения.
Шпаргалка: что значит каждое поле
Поле Обязательно? Что писать
id да Уникальный код латиницей, без пробелов, например p15. Для каждого блюда свой.
name да Название на сайте, как увидит покупатель.
price да Цена числом, без «₽», например 450.
unit да Как считается цена: 6 шт, за 1 кг и т.д.
image желательно Ссылка на картинку в интернете. Если не уверены — попросите разработчика залить фото и дать ссылку.
category нет Группа для себя, на сайте можно использовать позже. Например «Блины».
description нет Короткое описание под названием.
Если не сохраняется или «неверный ключ»
  • Проверьте, что ключ скопирован полностью, без пробела в начале или конце.
  • Убедитесь, что сайт запущен на сервере (без этого кнопка «Сохранить» не сработает).
  • Если пишет про JSON — где-то лишняя или пропущенная запятая/кавычка; сравните с соседними блоками.
  • В спорном случае пришлите разработчику текст из поля или скрин ошибки.

Важно: не удаляйте кавычки " и запятые между блоками. Перед большими правками скопируйте весь текст в «Блокнот» — так будет запасная копия.
