Инструкция для владельца

Страница только для вас — в меню сайта не отображается. Сохраните ссылку /help в закладки.

Как принять заказ

  1. Заказ приходит в Telegram.
  2. Зайди в админку /admin.
  3. Найди заказ, свяжись с клиентом.
  4. Измени статус: Готовится → В пути → Доставлен.
Открыть админку

Как управлять каталогом

  1. Зайди в админку /admin → вкладка Каталог.
  2. Нажми на фото, чтобы загрузить своё.
  3. Меняй название, цену, описание.
  4. Нажми Сохранить.

Как добавить новый товар

  1. Зайди в админку → Каталог.
  2. Нажми кнопку «Добавить товар» внизу.
  3. Заполни поля и загрузи фото.
  4. Сохрани.
